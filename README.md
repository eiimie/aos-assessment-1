## Task 1  
The file for Task 1 is **task1.sh**. This was developed and tested on macOS and Kali Linux (VirtualBox).

### Running on Linux/macOS
1. Open terminal and navigate to the script location:
```cd ~/Desktop```

2. Make the script executable:
```chmod +x task1.sh```

3. Run the script: 
```./task1.sh```
